# Industry Readiness Through Real-World Data Streaming with Apache Kafka

A hands-on, industry-focused workshop introducing students to real-time data streaming, event-driven architectures, and production-level engineering workflows using Apache Kafka.

## Objectives
- Equip students with practical, industry-grade knowledge of event-driven architectures and real-time data streaming using Apache Kafka.
- Closing the gap between academic exposure and modern software engineering practices used in tech companies
today.

  
## Outcomes
- Participants will simulate real-world industry use cases such as fraud detection pipelines, log streaming, microservices communication, and distributed messaging.
- Participants will build an end-to-end Kafka-based streaming pipeline.
- Participants will work hands-on using either institute-provided thin clients in Academic Block 4 or their own systems.
